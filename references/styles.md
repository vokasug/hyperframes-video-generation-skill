# Пресеты стиля

Каждый пресет: палитра с ролями, шрифты (все — с кириллицей, из бандла скилла), моушн-язык,
правила «не делать». Пресет выбирается в вопросах; «свой стиль» = пользователь даёт HEX-цвета
и (опц.) логотип — собери по образцу Neon Glass.

## 1. Neon Glass (дефолт; тёмный, технологичный)

- Фон: `#0A0D1C`; точечная сетка `radial-gradient(rgba(255,255,255,.07) 1.5px, transparent 1.6px)` 44px;
  2–3 размытых орба (blur 90–110px, opacity .3–.5) фирменных цветов, медленный дрейф
- Акценты (по смысловым блокам): violet `#8B5CFF`, cyan `#3ED3FF`, pink `#FF5CA8`,
  orange `#FF9A5C`, green `#4ADE80`, yellow `#FFD166`, red `#FF5C6C`
- Текст: основной `#F4F6FF`, приглушённый `#9AA3C7`
- Карточки: `rgba(255,255,255,.06)` + border `rgba(255,255,255,.14)`, radius 24–32px,
  тень `0 20px 60px rgba(0,0,0,.35)`
- Шрифты: Unbounded 700/900 (заголовки, термины), Manrope 400/600/800 (текст), JetBrains Mono (код/цифры)
- Градиентный текст: `background: linear-gradient(92deg,A,B); -webkit-background-clip: text; color: transparent`
- Моушн: входы `back.out(1.3–2)` scale/y, fade сцен 0.4–0.45с, микрокачания `sine.inOut` yoyo,
  glow-пульсы. Прогресс-бар снизу 6–7px градиентом, scaleX 0→1 линейно за всё видео.
- Не делать: светлый фон, засечные шрифты, больше 4 акцентов в одной сцене.

## 2. Light Minimal (светлый, деловой)

- Фон: `#FAFBFF`; тонкая сетка линий `rgba(20,30,60,.05)`; без орбов, максимум одно пятно
  градиента `#E8EEFF`
- Акцент: один основной (напр. indigo `#4F5DFF`) + один тёплый (`#FF8A5C`) для CTA;
  остальное — графит `#1B2233` и серый `#6B7488`
- Карточки: белые, border `#E4E8F2`, тень лёгкая `0 8px 30px rgba(30,40,80,.08)`
- Шрифты: Unbounded 700 (кратко, крупно), Manrope (всё остальное), JetBrains Mono (данные)
- Моушн: сдержанный — fade+y 20–30px, `power3.out`, без back.out-пружин кроме CTA
- Не делать: неон, тяжёлые тени, больше 2 акцентных цветов.

## 3. Bold Poster (громкая типографика, промо)

- Фон: чередование сцен — тёмный `#0E0E12` и кислотный `#D6FF3F` / `#FF5CA8` плашками
- Акценты: чёрный + один кислотный на сцену; текст до 160px, uppercase, Unbounded 900
- Карточки: жёсткие плашки без blur, толстые рамки 4–6px, offset-тени (8px, solid)
- Шрифты: Unbounded 900 всюду; Manrope только для мелких подписей
- Моушн: резкие входы (y 100%, `power4.out`, 0.35с), штампы с `back.out(2)`, rotation ±2–4°
- Не делать: градиенты в тексте, мелкий кегль >20% кадра, больше 12 слов на сцене.

## 4. Corporate Clean (сдержанный, внутренние обучалки)

- Фон: `#0F1420` или `#FFFFFF` по бренду; один бренд-цвет (попроси HEX, дефолт `#3E7BFA`)
- Акценты: бренд + нейтральные `#8A93A6`, `#22C55E` для «ок», `#EF4444` для «стоп»
- Карточки: radius 16px, минимум декора, иконки-линии 1.8px
- Шрифты: Manrope всюду + Unbounded 700 только для терминов; JetBrains Mono для команд/кода
- Моушн: самый спокойный — fade, y 20px, без пульсаций; переходы сцен — простое перекрытие-fade
- Не делать: кислот, больше 1 акцента, emoji, пружинные ease.

## Общие правила стиля (любой пресет)

- Сетка 1920×1080 (или 1080×1920 / 1080×1080 по формату); безопасные поля ≥ 90px.
- Термины — всегда самым крупным кеглем на сцене (80–130px).
- Контраст текста к фону — проверяй глазами на снапшотах (check считает WCAG AA).
- Один смысловой блок = один акцентный цвет, сквозной через всё видео
  (напр. «модель = violet, агент = pink, настройки = cyan»).
