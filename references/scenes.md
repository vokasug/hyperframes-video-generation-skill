# Сцены-архетипы (проверено реальным рендером)

Каталог готовых паттернов. Каждый: назначение, компоновка, анимация, подводные камни.
Тайминги внутри сцены привязывай к меткам предложений из timeline.json (srt_timeline.py).
Общие правила движения — в `hyperframes.md` (fade-обёртка сцены, hard-kill, разрешённые свойства).

## title-hero — титульная сцена
Плашка-бейдж сверху → заголовок по словам (y 90, `back.out(1.4)`, stagger 0.2) → подзаголовок →
пилюля «для кого» (scale .5→1, `back.out(2)`) → 3–4 парящих икон-чипа по углам (scale 0 → float
yoyo ±14px, sine 1.5с). Искра-звёздочки (rotation+scale yoyo). Выход: вся сцена y −30 + fade.

## level-stack — стопка уровней/частей
Заголовок сверху. 3 карточки 980×152 в колонку (номер 01/02/03, иконка в градиентном квадрате,
имя Unbounded 46px, теглайн справа). Появление снизу (y 170, stagger под слова озвучки),
шевроны между уровнями, финал — последовательный pulse scale 1.035 каждой карточки.
Камень: карточки ставь в обратном порядке построения (верхняя = последний уровень).

## neural-network — нейросеть/мозг
SVG генерируется скриптом: 3 колонки узлов (4-5-3), рёбра все-со-всеми (stroke rgba .28),
узлы pop `back.out(2)` stagger 0.045. 3–4 сигнальных шарика (r 8, glow) едут input→hidden→output
двумя линейными твинами по `attr:{cx,cy}`, repeat 6, stagger 0.55. Справа чипы-факты (x 60 → 0).
Низ: мини-чат «запрос → ответ» двумя пузырями.

## token-split — разбиение на части (токены/слова/этапы)
Термин крупно слева сверху. Фраза в пузыре → стрелка вниз → ряд цветных чипов-моношрифтом
(y 34 + scale .5, `back.out(1.8)`, stagger 0.13) → плашка-вывод («7 токенов · 1 ≈ 3–4 символа»).

## context-window — ограниченная ёмкость (память/лимит/бюджет)
Окно-карточка с шапкой (название + счётчик). Счётчик 0→N твином объекта + onUpdate
(`Math.round(v).toLocaleString("ru-RU")`). Внутри 4–6 пузырей-сообщений (скелетные линии)
въезжают справа (stagger 0.3); у лимита 1–2 старых падают влево-вниз (x −280, y 110,
rotation −8, fade), остальные сдвигаются. Плашка-предупреждение снизу + pulse.

## hallucination-stamp — ошибка и разоблачение
Треугольник-warning + термин. Пузырь «уверенный ответ» → красный штамп поверх
(scale 2.4→1, rotation −22→−10, `back.out(1.6)`) + приглушение пузыря (opacity .45) →
зелёная карточка «правильно: …» с галочкой → баннер-призыв снизу (pulse yoyo).

## robot-arms — агент/инструмент с «руками»
Бейдж уровня + термин слева. Робот по центру-слева: голова (скругл. прямоуг., антенна с шариком,
два глаза-glow, рот-полоска) + тело с иконкой; pop `back.out(1.5)`, float yoyo, моргание глаз
(scaleY .15, 0.09с, repeatDelay 1.7). Руки — SVG-пути к 3 карточкам-действиям справа,
рисование `strokeDashoffset 400→0` 0.6с + карточки x 80→0. Низ: ряд чипов-примеров
(контейнер целиком fade-in перед stagger — иначе подпись видна раньше времени!).

## settings-grid — панель настроек
Слева карточка-панель (2–3 слайдера: label, значение mono, ручка-glow, виляние x ±26 yoyo).
Справа сетка 2×2 карточек (иконка, тайтл Unbounded 37px, описание 1 строка — проверяй
переполнение: max ~28 символов при ширине 470px/26px шрифт!). Появление pop `back.out(1.7)`
строго по меткам озвучки. Под сеткой — чипы-примеры (stagger 0.12).

## shield-block — безопасность/блокировка
Термин + большой щит (fill 12%, drop-shadow glow, pop + pulse). Атака: чип-конверт летит
с края к щиту (x твин, `power2.in`) → щит вспыхивает (scale 1.18, 0.22с yoyo, `overwrite:"auto"`)
→ конверт отскакивает (rotation 25, opacity .25) → подпись «заблокировано». Низ: 3 карточки-меры.

## cta-map — финал: рекап + призыв
Верх: мини-карта пройденного (3 чипа со стрелками, stagger 0.35). Центр: крупная кнопка
(градиент, radius 999, glow box-shadow, pop `back.out(1.5)` + pulse 1.045 yoyo) + чип с именем
файла/ссылкой (mono). Низ: финальная фраза с градиентным акцентом. 2–3 звёздочки-искры.
Выход: fade всей сцены 0.6с.

## quote-card — цитата/мораль
Огромные кавычки-фигуры, текст цитаты 48–60px по центру, автор mono справа внизу.
Вход: слова stagger y 30 (0.03). Фон: орб акцента за цитатой.

## big-number — ключевая цифра
Число 200–280px Unbounded 900 с count-up (объект + onUpdate), подпись под ним, контекст
мелко. Рядом 2–3 мини-карточки сравнения («было/стало») — стрелки роста/падения.

## flow-diagram — процесс/конвейер
3–5 узлов-чипов в ряд/змейкой, соединительные линии рисуются dashoffset по очереди,
по узлу пробегает «пакет»-шарик (linear x/y). Хорош для «как это работает».

## dossier-card — досье персонажа/героя (книги, истории, биографии)
Слева карточка ~620×800: бейдж «ДОСЬЕ» mono, круглый портрет-силуэт (SVG лайнарт 3–5 путей,
цвет героя), имя Unbounded 44px, подпись mono. Справа колонка чипов-фактов (по ~94px высоты)
— pop `back.out(1.5)` x 80→0 под реплики озвучки. Финальный факт — «золотой» чип (рамка/цвет
акцента). Вариант конфликта: крупный outline-термин («ИДИОТ?») → перечёркивание (scaleX 0→1
красной линии) → замена градиентным словом («ЧИСТАЯ ДУША»). Камень: чипы длиннее ~40 символов
переносятся — сокращай формулировки.

## relation-map — карта связей персонажей (сюжеты, команды, конфликты)
SVG поверх: узлы = круг r 46–66 (fill 12% цвета, stroke 2.5, иконка-лайнарт + имя Unbounded
27px + роль mono под кругом), pop `back.out(2)` по меткам появления в рассказе. Рёбра —
`<line>` с реальной длиной в `stroke-dasharray` (считай `Math.hypot` из координат!), твин
`strokeDashoffset L→0` 0.7с + плашка-подпись отношений («одержимость», «любовь») на середине
ребра (rect + text, opacity 0→1). Кульминация: pulse всей карты (scale 1.025, transformOrigin
центра) + рёбра `strokeOpacity .28→.95`. 4–5 узлов максимум, раскладку фиксируй вручную.
Камни: подписи узлов ставь ПОД кругом (над — залезают на бейджи сверху); плашки рёбер не
должны пересекаться с подписями узлов — разноси по вертикали минимум на 50px.

## count-up-drama — большая цифра + драматическое действие (деньги, ставки, потери)
Число 140–170px Unbounded 900 с count-up (объект + onUpdate, `toLocaleString("ru-RU")`),
подпись mono. Рядом «сцена действия» (камин, шредер, старт): SVG с loop-анимацией (пламя —
3 слоя path, `scaleY .9→1.12` yoyo, transformOrigin 50% 100%; искры — 6–8 кружков y −130 +
fade, repeat по остатку сцены). Объект-носитель (купюра, чип) pop → полёт в точку действия
(`power2.in`, rotation −25) → вспышка (fullscreen radial overlay opacity 0→.35→0 за 0.5с) +
усиление пламени (scale 1.28, 0.22с). Финал сцены — слэм-вывод 90–96px (scale 2.2→1,
`back.out(1.6)`, rotation −2°). Камни: числу и слэму — `white-space:nowrap` + явный width
(«100 000 ₽» переносится по узкому nbsp!); слэм центруй с запасом — при scale 2.2 он
выпирает за края.

## Универсальные приёмы

- Переход между сценами: fade-обёртка `.scene-fade` (in 0.45 / out 0.4–0.6) поверх общего фона.
  Для Bold Poster допустим «срез»: сцена уезжает y −100% c `power4.in`.
- Иконки: inline SVG 24 viewBox, stroke 1.8–2.2, round caps (стиль Lucide) — рисуй руками,
  без внешних библиотек.
- Счётчики: твин объекта `{v:0}` + onUpdate → textContent с `toLocaleString("ru-RU")`.
- Ничто не должно быть видно «раньше времени»: всё, что появляется по метке, — через fromTo;
  контейнеры с частично анимируемыми детьми прячь целиком до метки.
