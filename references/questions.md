# Вопросы пользователю

Задавай ОДНИМ блоком после изучения источника. Каждый вопрос — с дефолтом и вариантами.
Пропускай то, что пользователь уже указал в запросе. Разрешён ответ «всё по умолчанию» —
тогда бери дефолты и сразу показывай бриф.

## Блок вопросов (шаблон)

1. **Источник и фактура.** «Используем <файл/тему>? Если тема — собрать фактуру из интернета?»
   (если источник уже дан — не спрашивай)
2. **Цель и аудитория.** Кто смотрит (новички / коллеги / руководство / соцсети)? Что зритель
   должен унести? Тон: дружелюбный / деловой / вдохновляющий. Дефолт: дружелюбный, новички.
3. **Длительность.** Дефолт ~2 мин (≈240–260 слов озвучки). Вилка 1–3 мин. Предупреди:
   >3 мин — длинный рендер и риск потерять внимание.
4. **Язык и голос.** Дефолт: русский. Голос: мужской `ru-RU-DmitryNeural` (дефолт) /
   женский `ru-RU-SvetlanaNeural` / другой из `edge-tts --list-voices`. Темп: бодрый +5%
   (дефолт) / спокойный 0%. Если запрос пользователя НЕ на русском — обязательно уточни язык
   озвучки и голос.
   - **Формат подачи.** Монолог (дефолт) / **подкаст-диалог** — два голоса с ролями:
     «ведущая + эксперт» (вопрос-ответ, дефолт для разборов), «дебаты» (два равных в споре),
     «интервью». Пара голосов дефолт: `ru-RU-SvetlanaNeural` (ведущая) + `ru-RU-DmitryNeural`
     (эксперт); уточни, кто из них в какой роли. Для диалога — `references/podcast.md`.
5. **Стиль.** Предложи пресеты из `styles.md` с одной строкой описания каждого
   (Neon Glass / Light Minimal / Bold Poster / Corporate Clean) + «свой» (цвета бренда,
   логотип — попроси HEX/файлы). Дефолт: Neon Glass.
6. **Формат кадра.** 16:9 (дефолт) / 9:16 (Shorts/Reels) / 1:1.
7. **Фоновая музыка.** Варианты: (а) свой файл — попроси путь; (б) автоподбор агентом —
   инструментал без слов, жанры: lo-fi/chill (дефолт для обучалок), корпоративный upbeat,
   эмбиент-электроника, кинематографичный минимализм; источник — первая живая точка цепочки
   из `music.md` (probe перед выбором!); (в) без музыки. Громкость дефолт: «голос чётко
   поверх» (−13…−17 дБ под речью). Кандидат показывается пользователю до рендера.
8. **Визуал и метафоры.** Предложи 2–3 набора сцен-архетипов из `scenes.md`, собранных под
   источник («для каталога терминов — стопка уровней, нейросеть, токены…»). Пользователь
   выбирает/правит. Отдельно: полные субтитры к озвучке — да/нет (дефолт нет, только
   крупные термины).

## После ответов

Сформируй бриф (этап 2 pipeline.md) и раскадровку. Запроси подтверждение — это ревизия ①.

## Правила

- Не задавай вопросы, на которые уже отвечено в запросе («2 минуты», «голос женский»).
- Не задавай больше 8 пунктов; объединяй мелкое.
- Предлагай конкретные варианты, а не «какой стиль хотите?» — пользователю проще выбрать,
  чем придумать.
- Любой неответ = дефолт. Не блокируйся на уточнениях больше одного раунда.
