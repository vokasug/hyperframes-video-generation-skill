# Troubleshooting

## check / lint

| Ошибка | Причина | Фикс |
|---|---|---|
| `media_missing_id` (audio will be SILENT) | у `<audio>` нет `id` | добавь уникальный `id` каждому аудио |
| `gsap_exit_missing_hard_kill` | exit-твин сцены заканчивается на границе клипа без `tl.set` | добавь `tl.set("<sel>", {opacity:0}, <граница>)` сразу после exit-твина |
| `gsap_css_transform_conflict` | у элемента CSS `transform` и GSAP-твин на тот же элемент | убери transform из CSS, используй `fromTo` |
| `overlapping_gsap_tweens` | два твина на одно свойство одного элемента пересекаются | разведи по времени или `overwrite: "auto"` |
| `composition_file_too_large` (warn) | >500 строк | предупреждение; можно жить, или выноси сцены в sub-compositions |

## Картинка

| Симптом | Причина | Фикс |
|---|---|---|
| Пустой кадр на середине сцены | элемент виден с начала сцены, а появление анимировано позже | всё «позднее» — через fromTo; контейнер прячь до метки |
| Текст вылезает за край карточки | длинная строка | ~28 символов на 470px при 26px Manrope; сокращай текст, не мельчи шрифт |
| Большой текст переносится/рвётся («100 000 ₽») | shrink-to-fit у abs-позиционированного блока без width; узкий nbsp из toLocaleString | `white-space:nowrap` + явный `left`/`width`; форматируй числа и проверяй снапшот |
| Градиентный текст невидим (`text_not_painted`) | градиент + `background-clip:text` на родителе, а текст — в спанах-словах | градиент и `-webkit-text-fill-color:transparent` вешай на сами спаны |
| Панель/полоса «уехала» наверх кадра | дети панели с `top:` лежат прямо в полноэкранном клипе | детей — внутрь позиционированного контейнера полосы (`bottom:0;height:110px`) |
| Кадр за пределами видео (напр. 348с) в snapshot | repeat цикла посчитан от TOTAL при старте t0>0 — таймлайн длиннее видео | `repeat = ceil((TOTAL - t0)/d) - 1`; см. hyperframes.md правило 7 |
| Подпись/лейбл виден раньше stagger-анимации детей | контейнер не скрыт | fade-in контейнера на метку, stagger детей после |
| Шрифты не те (fallback) | fonts.css перенесён из assets/ | пути в fonts.css относительны файлу (`fonts/…`), держи в `assets/` |
| Двойная экспозиция на стыке сцен | exit одной сцены пересекается с входом другой по z | сцены на разных треках + fade-обёртки; стык ровно в границу |
| «Застрял» кадр прошлой сцены | нет hard-kill | см. выше |

## Рендер

| Симптом | Причина | Фикс |
|---|---|---|
| Видео короче задуманного | нет `data-duration` на руте; таймлайн короче | `data-duration` + `tl.set({},{},TOTAL)` |
| Нет звука | нет `id` у audio; mp3 не существует | id + `ls -lh assets/vo/` |
| Музыка громче голоса | `data-volume` высокий | 0.12–0.15; проверяй volumedetect |
| Рендер висит/долго | нормально: ~2.2× длины видео | жди; для ревизий `-q draft` |
| Разный результат от кадра к кадру | Math.random / wall-clock в JS | только фиксированные координаты и GSAP-таймлайн |

## Озвучка

| Симптом | Причина | Фикс |
|---|---|---|
| Аббревиатуры читаются криво | TTS читает латиницу по-английски | подстановки из tts.md (эл-эл-эм, эм-цэ-пэ) |
| Видео длиннее цели | много текста | режь текст, потом темп до +10% |
| edge-tts не найден | не установлен | `brew install pipx && pipx install edge-tts` |

## Окружение

| Симптом | Фикс |
|---|---|
| `node` старый (<22) | `brew install node` (или nvm) |
| `ffmpeg` нет | `brew install ffmpeg` |
| Источник музыки не открывается | нормально без VPN; `probe_music.sh` → следующий в цепочке |
| Скачивание музыки обрывается на ~15–150 КБ | сетевой фильтр режет большие ответы по хостам (Mixkit/Incompetech/archive.org — зафиксировано) | сверяй размер с Content-Length; обрыв → Wikimedia Commons (рецепт в music.md) |
