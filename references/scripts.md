# Нарративные шаблоны сценариев

Тип видео выводится из источника и цели (подтверждается пользователем в брифе).
Каждый шаблон — каркас сцен; длительности — доли от общего времени.

## 1. Эксплейнер / каталог / обучалка

| # | Сцена | Доля | Содержание |
|---|---|---|---|
| 1 | Хук-титул | 6–8% | Название + обещание («N минут — и разберёшься») |
| 2 | Карта темы | 6–8% | Структура целиком: «всё складывается из N частей» |
| 3..N-2 | Блоки | поровну | Один блок = 1–3 сцены: термин крупно → метафора → деталь → пример |
| N-1 | Риски/нюанс | 8–10% | «И главное» — предупреждение или тонкость |
| N | Рекап + CTA | 8–10% | Карта ещё раз мельком + что сделать дальше |

Приёмы: термин всегда крупным Unbounded; сложное — через метафору (мозг, руки, окно, щит);
примеры — конкретикой (имена, числа), не абстракцией.

## 2. Книга / статья / история

| # | Сцена | Доля | Содержание |
|---|---|---|---|
| 1 | Хук | 6–8% | Обложка/название + интрига («о чём эта книга на самом деле») |
| 2 | Завязка | 10% | Мир, герой, исходная ситуация |
| 3..N-3 | Сюжет/идеи | поровну | 2–4 ключевых поворота или идеи, каждая — сцена с метафорой |
| N-2 | Кульминация | 10% | Главный конфликт/инсайт |
| N-1 | Мораль | 10% | Что автор хотел сказать — крупная цитата |
| N | Выводы + CTA | 10% | 2–3 вывода для зрителя + «читать/не читать и кому» |

Приёмы: цитаты — отдельной сценой-карточкой; персонажи — именные чипы; сюжет — линия
нарастания (ступени/дуга). Спойлеры — спроси пользователя, допустимы ли!

## 4. Отчёт / данные

| # | Сцена | Доля | Содержание |
|---|---|---|---|
| 1 | Хук | 6–8% | Главная цифра крупно |
| 2 | Контекст | 10% | Что измеряем и зачем |
| 3..N-2 | Находки | поровну | Одна находка = сцена: число крупно + визуализация (рост/падение/доли) |
| N-1 | Сравнение | 10% | Было/стало или мы/рынок |
| N | Выводы | 10% | 2–3 следствия + что делать |

## Общие правила сценария

- Разговорный язык, короткие предложения (до ~15 слов), «ты» или «вы» — по тону брифа.
- Одна сцена озвучки = 1–3 предложения. Никаких абзацев.
- Подстановки произношения сразу в тексте (см. `tts.md`).
- Последняя сцена всегда даёт зрителю следующий шаг (CTA): открыть, прочитать, попробовать.
- Хук обязан заработать за первые 3 секунды: вопрос, дерзкое число или обещание.
